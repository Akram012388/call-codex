# Marketplace Copy

## Name

CALL-CODEX

## Short Description

Dial worker threads from Codex.

## Long Description

CALL-CODEX is a macOS-first Codex plugin for super-user multi-thread missions. A main thread can boot a managed loopback app-server, create worktree-isolated worker threads, assign first-class roles, send call-line messages, stream worker progress, cleanly remove threads, and preserve local SQLite receipts.

## Starter Prompts

- Call Codex with builder and reviewer workers.
- Open a worktree call for this repo.
- Show call status and worker progress.

## Capabilities

- Interactive
- Write
- MCP
- Local

## Safety Notes

CALL-CODEX is local-first. The managed app-server binds to `127.0.0.1`, call state is stored locally, and worker threads remain under explicit user control.
